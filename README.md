## How the docs are organized

- **Primary axes**
  - By **type**: Blocks / Items / Recipes / Mechanics
  - By **progression** (“era”): Stone / Bronze / Iron / etc. (inside each type as needed)
- **Cross-references everywhere**
  - Item pages link to their recipes and mechanics
  - Mechanic pages link back to the items/blocks they affect
