## Axes (early)

Axes are important early tools for woodworking systems like hewing.

## Basic axes

The tag `#materia:basic_axes` includes (see [tag reference](../reference/tags/early-crafting-and-woodworking.md#materiabasic_axes)):

- hand axe
- wooden axe
- stone axe

These are used for:

- [Hewing](hewing.md)

## All axes

The tag `#materia:all_axes` includes `#materia:basic_axes` plus higher-tier axes (like bronze axes) (see [tag reference](../reference/tags/early-crafting-and-woodworking.md#materiaall_axes)).

This is used for recipes like:

- sticks from rough planks
