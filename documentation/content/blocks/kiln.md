## Kiln

Kilns are used to heat and process items. Some kiln outputs (especially metals) can be **hot**.

## Variants and upgrades

- Kiln (basic)
- Chimney upgrade (advanced kiln)
- Bellows upgrade (higher temperature tier)

There are also specialized kiln blocks:

- Furnace kiln
- Blast furnace kiln

## Using tongs

If the output is hot metal, use tongs:

- Right-click to extract 1
- Shift + right-click to extract as many as possible
