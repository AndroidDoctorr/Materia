## Coke oven

The coke oven converts **coal** into **coal coke** using fuel.

## GUI

<img src="../../../shared/src/main/resources/assets/materia/textures/gui/coke_oven.png" alt="Coke oven GUI" width="512">

Slot layout:

- **Input**: coal
- **Fuel**: coal (the coke oven only accepts coal)
- **Output**: coal coke

## How it works

- **Input**: coal
- **Fuel**: coal (the coke oven only accepts coal)
- **Output**: coal coke

## Notes

Coal coke burns longer than coal/charcoal and is required for some high-heat kiln recipes.
