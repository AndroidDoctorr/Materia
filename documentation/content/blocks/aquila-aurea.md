## Aquila Aurea

The Aquila Aurea is a decorative metal statue/block.

It’s intentionally “later game” than stone decoration because it requires **metal plates** (anvil work).

## Crafting

- [Recipe JSON](../../../shared/src/main/resources/data/materia/recipes/aquila_aurea.json)

Recipe requirements (at a glance):

- `materia:gold_plate` (made via anvil metalworking)
- `minecraft:gold_ingot`

## Placement / facing

- Faces away from the player when placed (furnace-style).
