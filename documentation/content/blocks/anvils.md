## Anvils

Anvils are crafting stations for shaping **hot metal** into parts (rods, plates, tool heads, wire, etc.).

If you’re new to the system, start here:

- [Anvils (mechanic)](../../mechanics/anvils.md)

## Anvil tiers (at a glance)

| Tier | Slots | What it unlocks |
|---|---|---|
| **Stone anvil** | 1 tool + 1 input | Basic shaping and the “bootstrap” path into bronze tooling |
| **Bronze anvil** | 2 tools + 1 input | Recipes that need multiple tools (wire/drawplate/bore chains begin here) |
| **Iron anvil** | 3 tools + 2 inputs | Highest complexity recipes (multi-tool + sometimes multi-input) |
