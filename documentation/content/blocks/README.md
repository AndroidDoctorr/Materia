## Blocks

This section documents Materia blocks. Each block page should include:

- **What it is / why it exists**
- **How to obtain / craft**
- **Key interactions** (mechanics, right-click behavior, fluids, heat, etc.)
- **Related**: items/blocks + recipe JSON links

## Categories

- **Natural**: generated / found blocks (ores, trees, soils, etc.)
- **Craftable**: player-made blocks (pots, stations, machines)
- **By “era”** (optional): Stone / Bronze / Iron / etc.
