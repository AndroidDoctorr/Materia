## Items

This section documents Materia items. Each item page should include:

- **What it is / what it’s for**
- **How to obtain** (crafting, drops, processing)
- **How it’s used** (mechanics, right-click behavior, durability, etc.)
- **Related**: blocks/recipes/tags

## Categories

- **Natural**: gathered items (fibers, rocks, resins, etc.)
- **Craftable**: tools, components, processed materials
- **By “era”** (optional): Stone / Bronze / Iron / etc.
