## Axes

Axes are core woodworking tools.

## Early axes

- [Hand axe](../hand-axe.md)
- Wooden axe
- Stone axe

These are “basic axes” used by recipes like [Hewing](../../../mechanics/hewing.md).

## Tags

- `#materia:basic_axes`: hand axe, wooden axe, stone axe (see [tag reference](../../../reference/tags/early-crafting-and-woodworking.md#materiabasic_axes))
- `#materia:all_axes`: basic axes + higher tiers (e.g. bronze axes) (see [tag reference](../../../reference/tags/early-crafting-and-woodworking.md#materiaall_axes))

## Crafting with axes

Some recipes treat axes as tools (they survive crafting but take durability damage).

Example:

- [Any rough plank + any axe → sticks](../../../../shared/src/main/resources/data/materia/recipes/sticks_from_rough_planks_any_axe.json)
