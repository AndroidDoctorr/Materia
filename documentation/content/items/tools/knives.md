## Knives

Knives are cutting tools used in crafting and gathering.

## Knife tiers

- Flint knife (early)
- Bronze knife
- Iron knife
- Steel knife

## Tree interactions (sap / latex)

Some knife tiers can be used to harvest:

- sap from spruce logs
- latex from rubber tree logs
