## Latex

<img src="../../../shared/src/main/resources/assets/materia/textures/item/latex.png" alt="Latex" width="64" height="64">

Latex is gathered from rubber trees using early cutting tools.

## Gathering

Right-click a **rubber tree log** with a tapping-capable tool:

- rubber tree log → tapped rubber tree log
- drops 1–2 latex

More complete reference:

- [Rubber tree](../blocks/rubber-tree.md)
- [Tapping (sap and latex)](../../mechanics/tapping.md)

## Fire pit

Latex can be processed in a fire pit:

- [Latex → rubber sheet](../../../shared/src/main/resources/data/materia/recipes/latex_to_rubber_sheet.json)

Also, rubber logs themselves have a fire pit recipe:

- `shared/src/main/resources/data/materia/recipes/rubber_tree_log_to_charcoal.json` (yields charcoal + latex)
