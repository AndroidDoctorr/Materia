# Materia

This folder contains the main documentation for Materia.

## What is it?

**Materia** is a progression overhaul mod that replaces “one-step” vanilla advancement with a realistic, station-based tech tree: you begin with stone-age gathering, knapping, and woodworking (hewing), unlock a primitive crafting workstation, then move into ceramics, heat management, kilns, and tiered anvils where **hot-metal handling and tool-tagged forging recipes** matter; on top of that foundation it layers interconnected systems like liquids/fermentation (amphorae), boiling/processing (water pot), textiles and storage (sewn sacks), expanded dyes, and later-game mechanical/electrical and combat-adjacent content (logic blocks, power components, cannons).

If you need copy/paste-able blurbs (multi-paragraph / single paragraph / single sentence), see [Descriptions](Descriptions.md).
