## Tyrian purple

Tyrian purple is a high-cost dye made from murex shell products.

## Chain overview

1) **Extract gland** (cutting tools)

- [Murex shell + cutting tool → murex gland](../../../shared/src/main/resources/data/materia/recipes/murex_gland_extraction.json)

2) **Boil gland** (water pot)

The water pot can boil murex glands into boiled versions (see [Water pot](../../mechanics/../mechanics/water-pot.md)).

3) **Collect iota**

Boiled gland processing produces tyrian purple “iota” (intermediate).

4) **Make blob**

- [6× tyrian purple iota → tyrian purple blob](../../../shared/src/main/resources/data/materia/recipes/tyrian_purple_blob.json)

5) **Make dye**

- [6× tyrian purple blob → tyrian purple dye](../../../shared/src/main/resources/data/materia/recipes/tyrian_purple_dye.json)

## Uses

Tyrian purple dye is used to dye materials like wool clumps and to craft dyed textile items.

Examples:

- [Tyrian purple wool clumps](../../../shared/src/main/resources/data/materia/recipes/clump_of_wool_tyrian_purple.json)
